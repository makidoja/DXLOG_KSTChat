# DXLog KST Chat Bridge with AirScout — Version 2.1

Version 2.1 is the first formal release of the redesigned DXLog custom form for ON4KST and AirScout.

## Highlights

- Automatic AirScout opportunity scanning for visible KST stations.
- Sortable `AS` column showing `NOW`, `Xm`, `-`, or blank.
- Distance filter from All to 2000 km.
- Active and worked-band station information.
- Stable KST refresh that retains the current user list until a complete replacement arrives.
- CQ, directed custom messages and four right-click editable macros.
- AirScout path and live matched aircraft on the built-in map.
- Cached, double-buffered map rendering with improved resizing, dragging and label placement.
- Active stations green, inactive stations yellow, selected station red, and one dedicated home marker.
- DXLog-style setup and macro dialogs.
- Band-change refresh of the user list, map and AirScout queue.

## Build

Build `DXLogKstBridge.csproj` using **Release | x86** with .NET Framework 4.8 and the DXLog assemblies installed.

The assembly and file version are `2.1.0.0`.
